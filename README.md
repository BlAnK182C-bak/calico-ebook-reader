# Welcome to Calico!
Calico is (currently) planned to be an ebook reader app with support for Epub parsing and planned support for PDF and MOBI formats.

!!! *THIS IS NOT A FINISHED PRODUCT YET - IT IS BARELY A PRODUCT RIGHT NOW* !!!

This is very much in a proof of concept stage.

The application is written entirely in [Rust](https://rust-lang.org/) and aims to be a modern open source alternative to a lot of ebook readers which seem to limit the reader in some way shape or form.

With planned features such as:
- Support for EPUB, PDF, MOBI formats
- Cross platform support across Android, iOS, MacOS, Windows and Linux
- Cross platform progress sync: have the option to sync your app across multiple platforms so you can continue reading ebooks no matter where you are.
- A plugin system allowing users creative freedom and control over the app
- TUI mode, for the fellow nerds.
- Dictionary plug and play via APIs
- Hardcover integration for tracking books on [hardcover.app](https://hardcover.app/) - (If goodreads ever decides to actually make a public API again I am more than willing to add this integration too.)

and more planned...

This README is simply a starting point for people landing here, it will continue to have more updates come the future.

This is my first time writing Rust code to a production level. A lot of the code here is pretty bad, and I hope to optimize this even beyond the beta releases.

*Please note that this app is very much still a proof of concept and a lot of features are subject to change or not be released at all. I am aiming to build a beta release in the coming months (By July of 2026) and pray that my timings actually match :"))
